# telnet.qqey.net

"telnet.qqey.net" is a Telnet server written in Golang.

## Usage

In this project, it is recommended to use "air," which allows for hot reloading in Golang.

```sh
make gorun
# or
go run main.go
```

## Docker Usage

```sh
make build

make up
```
